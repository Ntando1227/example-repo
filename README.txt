Project Name: 

Minesweeper.py

Description: 

This Python program works with a grid that contains mines (#) and empty spaces (-). It checks each empty cell and counts how many mines are around it (including diagonals). The program then replaces each empty space with a number showing how many nearby mines there are, making the grid easier to understand.

Main Parts
•	Check position → makes sure we don’t go outside the grid
•	Count mines → looks around a cell and counts mines
•	Run Minesweeper → builds the final result
